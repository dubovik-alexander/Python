программка для вывода всех возможных комбинаци слов (существительных) из заданных букв
написал по просьбе жены, для игрушки составления слов из букв (когда уже устаешь, но так хочется поймать бабочку) :)
я решил поэксперементировать и вроде получилось не плохо.
words.exe - исполняемый файл
russian_nouns.txt - файл со всеми существительными на русском языке, но туда можно легко добавлять слова.
Изначально файл был взят с ресурса: https://github.com/Harrix/Russian-Nouns/tree/main/dist
в отдельной папке: source code - исходный код на Pyhton
